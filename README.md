# Artado Docs

Artado Docs, modern sistemlerimiz için kullanılan bir belge yönetim sistemidir. PHP ile geliştirilmiştir ve kullanıcı dostu bir arayüz, kapsamlı içerik yönetimi ve güçlü bir veri tabanı yapısı sunar. 

## Özellikler

- **Kategoriler**: İçeriklerinizi düzenlemek için esnek kategori yapısı.
- **Admin Paneli**: İçeriklerinizi ve site bilgilerinizi düzenlemek için Modern CMS admin sistemi.
- **Sayfalar**: Her kategori altında birden fazla sayfa oluşturma imkanı.
- **Gelişmiş İçerik Yönetimi**: Uzun metin içerikleri destekleyen, zengin içerik oluşturma arayüzü.
- **Responsive Tasarım**: Farklı cihazlarda mükemmel bir görüntüleme deneyimi.
- **Geliştirici Desteği**: PHP ile geliştirilmiş, kolayca özelleştirilebilir yapı.

## Kurulum

1. **Veritabanı Oluşturma**:
  schema.sql dosyanızı phpmyadmin ile içe sql koduyla veritabanınızı oluşturabilirsiniz.
  
Yapılandırma: db.php dosyasında veritabanı bilgilerinizi girin.
Tarayıcıda Açın: Uygulamayı başlatmak için tarayıcınızda uygun URL'yi açın.
